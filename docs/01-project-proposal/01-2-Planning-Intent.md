- **문제 인식 (Problem)**:
    1. **게임의 단절성**: 기존 끝말잇기는 '한방 단어(해질녘, 이리듐)'가 나오면 게임이 허무하게 종료(Game Over)되며, 쌓아온 데이터가 초기화되는 문제가 있음.
    2. **데이터의 한계**: 정해진 국어사전 내에서만 플레이해야 하므로, 신조어, 밈(Meme), 고유명사 등 창의적인 단어 활용이 불가능함.
- **해결 방안 (Solution)**:
    1. **Git Branch 모델 도입**: 게임 오버 대신 '새로운 분기(Branch)'를 생성하게 하여, 실패를 '**새로운 경로의 시작**'으로 전환하고 데이터의 연속성을 보장함.
    2. **AI Hybrid Validation**: 표준국어대사전 API의 한계를 넘어, LLM을 활용한 유연한 검증을 도입하여 사용자가 만들어가는 '**살아있는 사전**'을 구축함.
